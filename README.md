# 🐟 Z&Z Peixaria - Chatbot de Atendimento

Chatbot simples e profissional de atendimento ao cliente para a peixaria Z&Z, desenvolvido com **Panel** (Python).

### Funcionalidades
- Respostas rápidas sobre **preços**
- Informações de **localização** e endereço
- Horários de atendimento
- Como realizar pedidos via WhatsApp
- Interface moderna e responsiva
- Totalmente feito em Python (sem JavaScript)

### Tecnologias
- Python
- Panel (HoloViz)
- ChatInterface
