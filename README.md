# 🐟 Z&Z Peixaria - Chatbot de Atendimento

Chatbot simples e profissional de atendimento ao cliente para a peixaria Z&Z, desenvolvido com **Panel** (Python).

### Funcionalidades
- Respostas rápidas sobre **preços**
- Informações de **localização** e endereço
- Horários de atendimento
- Como realizar pedidos via WhatsApp
- Interface moderna e responsiva
- Totalmente feito em Python (sem JavaScript)

### Tecnologias
- Python
- Panel (HoloViz)
- ChatInterface

### Como rodar localmente

```bash
# 1. Clone o repositório
git clone https://github.com/SEU_USUARIO/zz-peixaria-chatbot.git
cd zz-peixaria-chatbot

# 2. Instale as dependências
pip install -r requirements.txt

# 3. Rode o chatbot
panel serve app.py --show
